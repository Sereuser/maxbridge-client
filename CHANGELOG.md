# Changelog

Все заметные изменения в этом проекте будут документироваться в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
и этот проект придерживается [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.2.0] - 2026-03-14

### Добавлено
- Интерактивное меню в test_cli.py для работы с чатами
- Метод `get_chat_messages()` для получения сообщений чата
- Подробная документация API в `docs/API.md`
- Примеры использования в `examples/`
- Файл CONTRIBUTING.md для контрибьюторов
- Кэширование пользователей при логине
- Правильное отображение названий чатов

### Изменено
- Улучшена обработка ошибок
- Обновлена структура документации
- Улучшен README.md

### Исправлено
- Исправлен импорт в test_cli.py
- Убрано тестовое сообщение из основного скрипта

## [0.1.0] - 2026-03-14

### Добавлено
- Базовый WebSocket клиент для MAX API
- Авторизация по токену и SMS
- Отправка текстовых сообщений
- Управление чатами и пользователями
- Загрузка и скачивание файлов
- Модели данных (User, Chat, Message)
- Система исключений
- Асинхронный контекстный менеджер
- Базовая документация

### Технические детали
- WebSocket соединение к `wss://ws-api.oneme.ru/websocket`
- JSON-RPC 2.0 протокол с кастомными полями
- Автоматическое keepalive
- Кэширование данных
- Обработка переподключений
